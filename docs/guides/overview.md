### Requirements ###
 - [Node.js]


### Tech Stack: ###

BAR Lobby uses:
 - [Electron]
 - [Vue 3]
 - [TypeScript]
 - [SCSS (sass)]
 - [PrimeVue Components]

### Recommended Environment ###


It is highly recommended to use [VSCode] for development, as it provides full, built-in TypeScript support, as well as useful extensions such as [ESLint] and [Volar].
